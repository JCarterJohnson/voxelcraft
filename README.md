# VOXELCRAFT

A single-file, from-scratch 3D voxel sandbox game — mine, build, and craft by day... but something hunts the woods at night.

### ▶ Play in your browser
**https://jcarterjohnson.github.io/voxelcraft/**

Desktop only (needs a keyboard + mouse). Click **New World** to start — that also turns on the sound and locks your mouse for looking around (press **Esc** to free it).

### Controls
- **WASD** move · **Space** jump · **Shift** sneak · **double-tap W** sprint
- **Mouse** look · **Left-click** mine/attack · **Right-click** place / use item
- **1–9** hotbar · **E** inventory · **C** craft · **hold F** flashlight
- Survive the day. Then survive the night.

Built as a single `index.html` (no game engine, no dependencies) with Claude.
